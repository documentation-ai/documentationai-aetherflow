# Is AetherFlow secure?

Security is our top priority at AetherFlow. We employ bank-level encryption, comprehensive compliance certifications, and enterprise-grade security practices to protect your data.

## Encryption & Data Protection

### Data at Rest
- **AES-256 encryption** for all stored data
- **Customer-managed encryption keys** (Enterprise)
- **Secure key rotation** and management
- **Multi-region data storage** options

### Data in Transit
- **TLS 1.3 encryption** for all network communications
- **Perfect forward secrecy** for enhanced security
- **Certificate pinning** for mobile applications
- **Secure API endpoints** with rate limiting

## Compliance Certifications

### SOC 2 Type II

### GDPR Compliance

### HIPAA Compliance (Enterprise Add-on)
- **Business Associate Agreement (BAA)**
- **PHI data handling** procedures

- **Enhanced security controls**
