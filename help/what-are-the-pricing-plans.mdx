## Understanding Executions

### What counts as an execution?

An execution is counted each time your workflow runs:

- **Email processing**: Each email processed = 1 execution
- **Scheduled workflows**: Each scheduled run = 1 execution
- **API triggers**: Each webhook/API call = 1 execution
- **Manual triggers**: Each manual run = 1 execution

### Execution limits and overages

**Free Plan:**
- 1,000 executions/month
- Hard limit - workflows pause when reached
- Resets on the 1st of each month

**Pro Plan:**
- 50,000 executions/month per user
- Soft limit with notifications
- Additional executions: $0.001 each

**Enterprise Plan:**
- Unlimited executions included
- Custom limits available if requested
- Dedicated infrastructure for performance

## Frequently Asked Questions

**Can I change plans anytime?**
Yes, you can upgrade immediately or downgrade at the next billing cycle.

**Is there a setup fee?**
No setup fees for any plan. Enterprise plans may include onboarding costs.

**Do you offer annual discounts?**
Yes, Pro plan offers 25% savings when billed annually.

**Can I add users to existing plans?**
Yes, Pro and Enterprise plans support adding users with prorated billing.
