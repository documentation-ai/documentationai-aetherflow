## Zapier Migration

Step-by-step guide to migrate from Zapier to AetherFlow.

<Tabs>
  <Tab title="Export from Zapier" icon="download">
    1. Log into your Zapier account
    2. Go to Settings > Personal > Export Your Data
    3. Request data export (may take 24-48 hours)
    4. Download the JSON export file when ready
  </Tab>

  <Tab title="Import to AetherFlow" icon="upload">
    1. Create your AetherFlow account
    2. Navigate to Settings > Import/Export
    3. Select "Import from Zapier"
    4. Upload your exported JSON file
    5. Review and map integrations
    6. Test imported workflows
  </Tab>

  <Tab title="Post-Migration Tasks" icon="check-circle">
    - Verify all integrations are connected
    - Test workflows with sample data
    - Update webhook URLs if necessary
    - Monitor execution logs for issues
    - Optimize prompts for AetherFlow's AI
  </Tab>
</Tabs>

## Microsoft Power Automate Migration

Convert Microsoft Flows to AetherFlow workflows.

<Expandable title="Migration Process">
1. **Export Flows**: Use Power Automate's export feature to download flow definitions
2. **Analyze Structure**: Review flow logic, conditions, and actions
3. **Recreate in AetherFlow**: Use natural language to describe flow logic
4. **Map Integrations**: Connect equivalent AetherFlow integrations
5. **Test Thoroughly**: Validate functionality with comprehensive testing
</Expandable>

<Expandable title="Common Challenges">
- Complex conditional logic may need rephrasing
- Custom connectors require manual recreation
- Approval workflows need redesign for AetherFlow's model
- Power Automate-specific actions may not have direct equivalents
</Expandable>
