## Account Creation and Onboarding

You begin by signing up for AetherFlow to access the dashboard. The platform guides you through a quick onboarding process, explaining core concepts like prompts and triggers. This ensures you understand how natural language drives automations without technical barriers.

<Callout kind="tip" collapsed="false">
  Use your work email for team collaboration features from the start.
</Callout>

## Installation and Setup

No software installation is needed; AetherFlow runs in the cloud. Access it via any modern browser. For API usage, generate an API key from settings.

<Steps>
  <Step title="Create Account" icon="user-plus" title-type="p">
    Visit aetherflow\.com and click Sign Up. Fill in your details and verify via email.
  </Step>

  <Step title="Dashboard Tour" icon="home" title-type="p">
    Complete the interactive tour to familiarize yourself with the interface.

    ```javascript
    // Generate API key example
    const apiKey = await fetch('/api/keys', { method: 'POST' });
    console.log(`Your key: ${apiKey}`);
    ```
  </Step>

  <Step title="First Integration" icon="plug" title-type="p">
    Go to Integrations tab and connect Slack. Authorize permissions for messaging.
  </Step>
</Steps>

## Building Your First Workflow

Describe your automation in plain English. For example, "When I receive an email, summarize it and post to Slack channel #team-updates."

<Tabs>
  <Tab title="Basic Workflow" icon="play">
    Use the builder to input your prompt. Preview the generated steps.

    <CodeGroup show-lines="true" tabs={[]}>
      ```javascript
      // API to create workflow
      await fetch('/workflows', {
        method: 'POST',
        body: JSON.stringify({
          name: 'Email Summary',
          prompt: 'Summarize emails and post to Slack'
        })
      });
      ```

      ```python
      # Python SDK example
      from aetherflow import Client
      client = Client(api_key='your_key')
      workflow = client.workflows.create(
          name='Email Summary',
          prompt='Summarize emails and post to Slack'
      )
      ```
    </CodeGroup>
  </Tab>

  <Tab title="Advanced Triggers" icon="zap">
    Set natural language triggers like "On new ticket in Zendesk."

    <Expandable title="Trigger Options" default-open="false">
      Choose from email, app events, or scheduled runs.
    </Expandable>
  </Tab>
</Tabs>
